# באג מספר 1 - לולאה אינסופית

הקוד הבא אמור לקרוא מספר מהמשתמש ולהדפיס את כל המספרים החל מהמספר הזה בסדר יורד, אבל כאשר ננסה להריץ אותו הוא יתקע בלולאה אינסופית. 
השתמשו בריצה צעד אחר צעד כדי להבין איפה הבעיה

```csharp
Console.Write("Enter a positive number: ");
int n = int.Parse(Console.ReadLine());

while (n >= 0)
{
    Console.WriteLine(n);

    if (n > 0)
    {
        n = n - 1;
    }
}
```

שימו לב, אחרי שמצאתם את הבעיה, נסו במקום צעד אחר צעד להשתמש בפקודת Pause (נמצאת ליד כפתור העצירה)
במקרה של לולאה אינסופית. לפעמים עדיף "להכנס" ללולאה האינסופית ולבדוק את הבעיה בצורה כזו.
