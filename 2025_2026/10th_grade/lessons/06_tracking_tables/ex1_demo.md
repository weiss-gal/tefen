# תרגיל דוגמא - טבלת מעקב 1

הנחיה: עקוב באמצעות טבלת מעקב אחרי הקוד הבא. 

```csharp
int i = 1; // line 1
while (i<3)
{
    Console.WriteLine(i);

    i++;
}
```

| 1   | 1   |         |       |

| 4   |     |         | 1     |
| 5   | 2   |         |       |
| 2   |     | true    |       |
| 4   |     |         | 2     |
| 5   | 3   |         |       |
| 2   |     | false   |       |
