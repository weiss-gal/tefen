# לולאת for (מונה)

## מה זו לולאת for?

לולאת `for` היא סוג של לולאה שמשמש כאשר אנו יודעים מראש כמה פעמים אנו רוצים לחזור על פעולה מסוימת. להבדיל מלולאת `while` שמתבססת על תנאי, לולאת `for` מתבססת על מונה שמתקדם בכל איטרציה.

לולאת `for` מורכבת משלושה חלקים עיקריים:
1. **איתחול המונה** - קובע את הערך הראשוני של משתנה המונה
2. **תנאי הלולאה** - קובע מתי הלולאה תמשיך לרוץ
3. **עדכון המונה** - מעדכן את משתנה המונה בכל איטרציה

---

## מבנה של לולאת for

```csharp
for (איתחול; תנאי; עדכון)
{
    // הקוד שירוץ בכל איטרציה
}
```

### דוגמה בסיסית

```csharp
for (int i = 1; i <= 5; i++)
{
    Console.WriteLine("Print number " + i);
}
```

הלולאה הזו תדפיס:
```
Print number 1
Print number 2
Print number 3
Print number 4
Print number 5
```

---

## הסבר מפורט על חלקי הלולאה

### איתחול המונה
```csharp
int i = 1
```
- יוצר משתנה חדש בשם `i` ומגדיר את הערך שלו ל-1
- המשתנה הזה קיים **רק** בתוך הלולאה

### תנאי הלולאה
```csharp
i <= 5
```
- בודק אם הערך של `i` קטן או שווה ל-5
- כל עוד התנאי הזה נכון, הלולאה תמשיך לרוץ

### עדכון המונה
```csharp
i++
```
- מגדיל את הערך של `i` ב-1 בכל סוף איטרציה
- זה קצר יותר מלכתוב `i = i + 1`

## דוגמאות שימושיות

### הדפסת מספרים זוגיים
```csharp
Console.WriteLine("Even numbers from 2 to 20:");
for (int i = 2; i <= 20; i += 2)
{
    Console.WriteLine(i);
}
```

### חישוב סכום
```csharp
int sum = 0;
for (int i = 1; i <= 10; i++)
{
    sum = sum + i;
}
Console.WriteLine("הסכום של המספרים מ-1 עד 10 הוא: " + sum);
```

### לולאה לאחור
```csharp
Console.WriteLine("ספירה לאחור:");
for (int i = 10; i >= 1; i--)
{
    Console.WriteLine(i);
}
Console.WriteLine("הקצב!");
```

## תרגילים לתרגול

### תרגיל 1: טבלת כפל
כתוב תכנית שמדפיסה את טבלת הכפל של המספר 7:
```
7 × 1 = 7
7 × 2 = 14
7 × 3 = 21
...
7 × 10 = 70
```

### תרגיל 2: חישוב עצרת (Factorial)
כתוב תכנית שמחשבת את העצרת של מספר שהמשתמש מכניס.
עצרת של 5 = 5 × 4 × 3 × 2 × 1 = 120

### תרגיל 3: כוכבים
כתוב תכנית שמדפיסה את התבנית הבאה:
```
*
**
***
****
*****
```

## השוואה בין לולאת while ללולאת for

### אותה פעולה עם while:
```csharp
int i = 1;
while (i <= 5)
{
    Console.WriteLine("Print number " + i);
    i++;
}
```

### אותה פעולה עם for:
```csharp
for (int i = 1; i <= 5; i++)
{
    Console.WriteLine("Print number " + i);
}
```

**מתי להשתמש בכל אחת?**
- **לולאת for**: כאשר יודעים מראש כמה פעמים צריך לחזור
- **לולאת while**: כאשר התנאי מורכב יותר או לא ברור כמה איטרציות יהיו

## טעויות נפוצות

### 1. שכחת עדכון המונה (לולאה אינסופית)
```csharp
// שגוי - הלולאה לא תיעצר!
for (int i = 1; i <= 5; /* שכחנו את i++ */)
{
    Console.WriteLine(i);
}
```

### 2. שימוש במונה מחוץ לטווח
```csharp
for (int i = 1; i <= 5; i++)
{
    Console.WriteLine(i);
}
// Console.WriteLine(i); // שגיאה! i לא זמין כאן
```

### 3. התחלה מהמקום הלא נכון
```csharp
// אם רוצים מ-1 עד 5, נתחיל מ-1 ולא מ-0
for (int i = 1; i <= 5; i++) // נכון
for (int i = 0; i <= 5; i++) // ייתן 6 איטרציות במקום 5!
```

## תרגיל בית

התרגיל הבא מתחיל בסיפור. 

באחד מהטיולים שלי פגשתי ביצור שקוראים לו ... אני לא ממש יודע איך מבטאים את השם שלו, אז אני קורא לו בליפ. תבינו משהו - בליפ הוא חייזר מכוכב שבתאי. בכוכב שבתאי, תאמינו או לא, גם נוהגים לחגוג את חג החנוכה. זה לא בדיוק החנוכה שלנו, אצלהם בסיפור הם נלחמו ב"קרוניים" במקום ביוונים, אבל העקרון דומה. 

העניין הוא ששנה על כוכב שבתאי נמשכת 10759 יום. 
אבל זה לא נגמר פה, גם הימים בשבתאי שונים, הם בערך 10.7 שעות שלנו. כתוצאה מכך שנה של שבתאי נמשכת 24,491 יום. לכן בהתאמה את חנוכה הם חוגגים במשך 536 יום. 
כפי שבטח כבר הבנתם, הנרות בחנוכה זה ביג דיל בהוצאות השנתיות שלהם. לכן החלטתי לעזור לבליפ ולכתוב לו תכנה שתחשב כמה נרות הוא צריך לכל חג החנוכה. אבל בגלל שאנחנו מהנדסים טובים, נכתוב תכנה שיכולה להתמודד עם כל סוג של "חנוכה" (יש לו גם חברים בכוכבים אחרים , בטח גם הם ירצו עזרה)

המשימה שלכם היא מאוד פשוטה :

כתבו תכנה הקולות מהמשתמש מספר אחד. המספר הוא האורך של "חנוכה" .
השתמשו בלולאת for כדי לחשב כמה נרות צריך להדליק בכל החג יחד. 

דוגמא, חג חנוכה בכדור הארץ נמשך 8 ימים, לכן חבילת נרות בישראל מכילה בדיוק 44 נרות.

שימו לב, אל תשכחו לספור את השַׁמָּשׁ, יש לנו אחד כזה בכל יום.

בונוס: בונוס מיוחד ינתן למי שמצליח למצוא (בנוסף, לא במקום) פתרון שלא דורש אף לולאה. 
(רמז - חברו את סכום הנרות ביום הראשון עם היום האחרון, אחר כך חברו את סכום הנרות ביום השני וביום הלפני אחרון וכן הלאה, מה משותף לכולם)
