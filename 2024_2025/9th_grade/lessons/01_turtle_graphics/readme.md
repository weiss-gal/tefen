# צבים

## התקנה של הספרייה turtle
נפתח את שורת הפקודה של חלונות (לוחצים על מקש מיקרוסופט פעם אחת ומקלידים "cmd") ונפעיל את הפקודה הבאה. 
```
pip install turtle
```

## תכנית דוגמא ראשונה
התכנית הבאה תפתח מסך חדש, תיצור "צב" אחד בשם t ותיצור קו ישר:

```
import turtle

window = turtle.Screen()
t = turtle.Turtle()

t.forward(100)
```
