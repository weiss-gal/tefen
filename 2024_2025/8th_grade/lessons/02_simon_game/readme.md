## התנסות
הכנסו ללינק הבא ושחקו במשחק:
https://freesimon.org/

## תכנון
מה ההתנהגות של המשחק ? 
התייחסו לנקודות הבאות:

- איזה צבעים ישנם במשחק
- כמה צלילים יש במשחק
- איך עובדים השלבים
- איך עובד הניקוד
- כמה זמן יש לשחקן להגיב
